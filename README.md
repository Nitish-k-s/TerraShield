## 🧠 The Problem

Invasive species cost the global economy **$423 billion per year**. Traditional field surveys detect outbreaks 8–12 months after they begin — by then, containment is exponentially harder.

**TerraShield detects outbreaks in 2–4 weeks.**

## ✨ How It Works

```
Citizen uploads geo-tagged photo
        ↓
EXIF extraction → GPS coords, device, timestamp
        ↓
🧠 Hindsight Agent Memory Bank recalls past sightings nearby
        ↓
Gemini 2.5 Flash Vision AI analyses the image
  + satellite NDVI context injected into prompt
  + agent memory context injected into prompt
        ↓
Classification: invasive-plant / invasive-animal /
  deforestation / wildfire / flood-risk / normal
        ↓
Risk score (0–10) + species tags + ecological summary
        ↓
Memory retained → next analysis in this area is smarter
        ↓
Live map updated + outbreak clusters detected
```

---

## 🔑 Key Features

### 🧠 Agent Memory (Hindsight Agent Memory Bank)
The AI **learns from every report**. Before analysing a new image, it recalls past sightings within 50km and feeds that context into Gemini's prompt. The more reports submitted, the smarter the analysis gets.

> *"3 previous Lantana camara sightings detected within 5km — avg risk 8.1/10. Outbreak cluster confirmed. Escalate immediately."*

### 🛰️ Satellite Vegetation Analysis
Sentinel-2 satellite data provides NDVI (vegetation index) for every GPS coordinate — corroborating or challenging the AI's visual classification with hard satellite evidence.

### 🗺️ Live Outbreak Map
Real-time Leaflet.js map showing all analysed reports. Automatic cluster detection groups reports within 5km radius — when 3+ reports of the same species appear, an outbreak cluster is flagged.

### 📊 District Intelligence Dashboard
District-level analytics: risk distribution, species × district matrix, 30-day trend charts, top risk zones, active alert table — all filterable by country → state → district.

### 📄 AI-Generated PDF Reports
Full ecological risk reports generated by Gemini with 6 sections: species overview, threat assessment, satellite analysis, outbreak likelihood, recommended actions, final risk conclusion.

### 🏆 TerraPoints Gamification
Users earn points for verified invasive species reports (10/20/30 based on confidence). Level system: Observer → Defender → Protector → Guardian.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Gemini API key ([aistudio.google.com](https://aistudio.google.com))

### Installation

```bash
git clone https://github.com/Nitish-k-s/TerraShield.git
cd TerraShield
npm install
```

### Environment Setup

```bash
cp .env.example .env.local
```

Edit `.env.local`:
```env
JWT_SECRET=your-secret-here
GEMINI_API_KEY=your-gemini-key
PLANET_API_KEY=your-planet-api-key  # for satellite NDVI
```

### Run

```bash
npm run dev
# Open http://localhost:7001
```

### Seed Demo Data

```bash
node scripts/seed-data.mjs
```

## 🧠 Agent Memory — How It Works

```
Report 1: Lantana camara at 11.62°N, 76.59°E → retained in memory
Report 2: Unknown plant at 11.63°N, 76.60°E (500m away)
  → Agent recalls Report 1
  → Hindsight Bank summarizes: "Previous Lantana outbreak 500m away, risk 8.2/10"
  → Gemini receives this context → more accurate, escalated response
  → Report 2 retained → memory grows
```

Every analysis makes the next one smarter. This is the core innovation.
